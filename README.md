# AX2012 Sample Projects

This repository contains sample projects developed in Microsoft Dynamics AX 2012, focusing on real-world business scenarios and ERP customizations.

##  Project Overview

The purpose of this repository is to demonstrate practical experience in:

* X++ development
* SSRS report design and implementation
* Business process modeling within AX 2012
* Data handling using tables such as InventTrans and ledger entries

##  Included Modules

### 1. Insurance Process Module

* Tracks insurance coverage for warehouse products
* Handles insurance start and end dates
* Identifies products requiring renewal based on stock availability
* Implements status logic (Waiting, Process, Renewal Needed, Done)

### 2. Warehouse Expense Allocation

* Allocates indirect expenses across products
* Links ledger entries with sales and purchase transactions
* Supports monthly reporting and cost analysis

### 3. SSRS Reports

* Trader profitability report
* Warehouse rent expense report
* Custom reports based on business requirements

##  Technologies Used

* Microsoft Dynamics AX 2012
* X++
* SSRS (SQL Server Reporting Services)
* SQL Server

##  Purpose

This is a learning and portfolio repository designed to:

* Practice GitHub and version control
* Showcase ERP development experience
* Provide structured examples of AX 2012 customizations

##  Note

All data and examples in this repository are for demonstration purposes only. No real company data is included.
